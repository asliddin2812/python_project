Python dasturlash tili tarixi

Python dasturlash tili 1980-yillarning oxirida Niderlandiyalik dasturchi Guido van Rossum tomonidan yaratilgan. Guido van Rossum o‘sha paytda ABC dasturlash tilida ishlagan bo‘lib, u tilning soddaligi va foydalanish qulayligidan ilhomlanib, undan ko‘ra kuchliroq va moslashuvchanroq til yaratishga qaror qiladi. 1989-yilning dekabr oyida, ta’til vaqtida, u yangi til yaratishni boshlaydi va unga “Python” deb nom beradi. Bu nom mashhur ingliz komedik shousi “Monty Python’s Flying Circus”dan olingan bo‘lib, haqiqiy ilon bilan bog‘liq emas. Python dasturlash tilining birinchi versiyasi, ya’ni Python 1.0, 1991-yilda chiqarilgan. U string, list va dictionary kabi asosiy ma’lumot turlarini, shuningdek funksiyalar, xatoliklarni ushlash (error handling) va asosiy modullarni o‘z ichiga olgan. Til sodda sintaksis va oson o‘qiladigan kod bilan ajralib turgan. 2000-yilda Python 2.0 versiyasi chiqarildi. Bu versiyada ko‘plab yangiliklar joriy etildi, jumladan avtomatik xotirani boshqarish (garbage collection), list comprehension’lar va Unicode qo‘llab-quvvatlovi. Ammo Python 2 va 3 versiyalari orasida orqaga moslik yo‘q edi, bu esa kelajakda muhim muammo bo‘lishi mumkin edi. 2008-yilda Python 3.0 chiqarildi, bu versiya Python tarixidagi katta burilish bo‘ldi. U tilni soddalashtirish va zamonaviy dasturlash talablari asosida qayta loyihalashtirishni maqsad qilgan. Muhim o‘zgarishlar qatoriga print funksiyasining sintaksisi, range() funksiyasining optimallashtirilishi va string’larning Unicode bo‘lishi kiradi. Bu o‘zgarishlar ba’zi eski kodlarning Python 3’da ishlamasligiga olib keldi, ammo tilning kelajagi uchun zarur edi.
